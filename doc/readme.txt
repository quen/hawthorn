This folder contains documentation and related files.

- hawthorn.odt, .pdf

OpenOffice document and resulting PDF file of the Hawthorn system manual.
Some of the images used in the manual are also included.

- connectordevelopment.odt, .pdf

OpenOffice document and resulting PDF file of the connector development guide.

- hawthorn.svg

Inkscape SVG file containing several variants of the Hawthorn logo.
[Requires Museo fonts.]

- Hawthorn logo PNGs

For use by any system that embeds Hawthorn.

- Readme files

These readme files are copied into various folders of the distribution.

- startupscripts folder

Platform-specific scripts to help run the Hawthorn server. Copied into the
distribution.
